# SD9 Pistol Frame and DIY Rail System

#### Author
FreeMenDontAsk

#### Version
1.0

#### Description
This the SD9 Pistol Frame and DIY Rail System by FreeMenDontAsk. The frame uses a DIY front rail which you make yourself; see the specsheets. This frame is inspired by S&W SD9.

Note: The SD9VE is gun as the SD9 but but made of stainless steel.

#### Print Settings
* 99% infill, quality PLA or ABS.

#### Parts Required
* SD9 or SD9VE parts kit
* Front DIY rail
* 1/8 roll pin

#### Assembly
* Same as standard SD9 or SD9VE
* Some hand filing required
