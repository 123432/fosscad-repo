# Glock G43 SS80 Frame 

#### Author
FreeMenDontAsk 

#### Description
this is the G43 SS80

This is a (Glock) G43 SS80 Frame (Lower) by Freemandontask. To complete the lower you will need the SS80 Builders Tool Set found [here](https://www.glockstore.com/SS80-Build-Set). This frame was inspired by the Glockstore SS80 80% Lower.

Make your own Single Stack Frame that you complete with Glock 43 parts. This SS80 Frame is compatible with parts designed for the Glock 43.

#### Parts Required
* G43 OEM Parts Kit
* SS80 Builders Kit

#### Print Settings
* 99% infill printed upside down with support

##### Assembly
* Some hand filling and fitting required
* Assemblies like a standard SS80 G43
